2. Javascriptövning

  • Kolla i console vad som skrivs ut när du klickar på knappen.
  • Uppdatera Notification.setNotification i notification.js så att texten på notifieringsytan byts ut mot texten från textfältet när man klickar på knappen.
  • Per kan visa exempel på script injection :)

3. SASS-övning

  • Hitta var stilen kommer ifrån som gör notifieringytan gul
  • Byt färg på notifieringen till error-röd (variabel finns i _colors.scss)
  • Byt färg på texten på ytan till vit (variabel finns)
  • Hitta stilen där varningsikonen sätts.
  • Gör en ny klass som heter <komponentnamnet>-error som använder icon_warning.svg som bild istället för den gråa bilden.
  • Lägg till klassen på notifieringselementet i index.html så att ikonen blir vit.
